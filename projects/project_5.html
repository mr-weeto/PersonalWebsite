Software Project: Paste Buddy Data Entry Tool

This tool was designed to make me more efficient at my job at a call center. I noticed that there were unecessary portions of time spent typing the same phrases over and over to customers in live chats and emails. There were canned responses and email templates that could be used, but the interfaces we used varied in usefulness across platforms. I created something that was portable, and suited the specific needs of that job and my coworkers quickly took a liking to the tool. The tool was written with vanilla CSS, HTML, and JavaScript so it could simply be emailed and saved as an HTML file on the computer. Last I checked, the department still uses the tool on a day to day basis.

The code is included below and will function, but behaves a little wonky on my personal website. You simply right click to save whatever is in your clipboard into a button, left click to copy it into your clipboard. Middle mouse button toggles the button to add a the current date and optional initials at the end of the text. Feel free to download the source code from my GitHub account.
