Python data entry tool

At my call center job my team was assigned busy work for our down time. The task was taking a report from one content management system that contained all of our old customer order data and manually entering it into a web form for the new system we planned to migrate to. We had been promised new quality of life improvements for our workflow once the task was completed so I saw no reason to waste any time finishing the task. I hacked together a python automation tool that took control of the mouse and keyboard, copied data from the spreadsheet containing the order data, pasted it into the web form, submitted it, then copied the data back into the spreadsheet in a different set of columns for later verification. I tested the script when I would otherwise be doing the busy work myself and fixed any problems with it. Once it was ready to launch I left it running over the weekend. This unfortunately backfired as the promised features were a bluff and the lack of extra busy work ended up with my teams hours being cut.
